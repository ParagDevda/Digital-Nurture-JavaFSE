# Week 2 - Spring Core, Maven, Spring Data JPA, and Hibernate

## Overview
This week focuses on the foundation of Spring-based Java applications and database integration.

## Topics Covered
- Spring Core
  - Dependency Injection (DI)
  - Inversion of Control (IoC)
  - Spring Beans and Bean Lifecycle
  - Configuration using XML and annotations
- Maven
  - Project structure
  - Dependencies management
  - Build lifecycle
  - Running Maven commands
- Spring Data JPA
  - Repository interfaces
  - CRUD operations
  - Derived query methods
  - Spring Data repository support
- Hibernate
  - ORM concepts
  - Entity mappings
  - Relationships between entities
  - Session and transaction management

## Learning Goals
- Understand the core concepts of Spring Framework
- Build Java projects using Maven
- Connect applications with databases using Spring Data JPA
- Use Hibernate for object-relational mapping

## Practice Tasks
- Create a Spring project using Maven
- Configure Spring beans and dependency injection
- Create JPA entities and repositories
- Perform CRUD operations using Spring Data JPA and Hibernate

## Notes
- Practice writing small projects to understand each concept clearly.
- Focus on how Spring manages objects and database interactions efficiently.
