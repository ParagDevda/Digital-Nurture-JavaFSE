# Week 1 - Deepskilling

## Overview
Week 1 focuses on fundamental concepts in Java Full Stack Development, covering data structures, design patterns, database management, and testing methodologies.

---

## Modules

### 1. Data Structures and Algorithms (DSA)
- Introduction to fundamental data structures
- Algorithm analysis and complexity
- Arrays, Linked Lists, Stacks, and Queues
- Sorting and searching algorithms
- Problem-solving with DSA

### 2. Design Patterns and Principles

#### SOLID Principles
- **S**ingle Responsibility Principle (SRP)
- **O**pen/Closed Principle (OCP)
- **L**iskov Substitution Principle (LSP)
- **I**nterface Segregation Principle (ISP)
- **D**ependency Inversion Principle (DIP)

#### Hands-On Design Patterns
- **Factory Design Pattern**
  - Simple Factory
  - Factory Method
  - Abstract Factory
  - Implementation and use cases

- **Singleton Pattern**
  - Thread-safe singleton
  - Lazy initialization
  - Implementation approaches
  - Best practices

### 3. PL/SQL
- Introduction to PL/SQL programming
- Basic syntax and structure
- Variables and data types
- Control structures (IF, LOOP, CASE)
- Procedures and Functions
- Error handling and exception management
- Cursors and bulk operations
- Database interactions and queries

### 4. Testing Using JUnit and Mockito

#### JUnit Framework
- Unit testing basics
- Test annotations (@Test, @Before, @After, etc.)
- Assertions and test cases
- Test suites
- Parameterized tests

#### Mockito Framework
- Mocking objects and dependencies
- Stubbing method calls
- Verification of interactions
- Mock spy objects
- Integration with JUnit
- Real-world testing scenarios

---

## Learning Outcomes
By the end of Week 1, you will:
- Understand and apply DSA concepts
- Design robust applications using design patterns
- Write and optimize SOLID principle-based code
- Create database operations with PL/SQL
- Write comprehensive unit tests with JUnit
- Mock external dependencies with Mockito
